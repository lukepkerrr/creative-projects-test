## Структура репозитория
* images - папка с изображениями
* index.html - страница с вопросом
* results.html - страница с результатами
* styles.css - стили
* question-script.js - файл скриптов для index.html
* results-script.js - файл скриптов для results.html
* interactive-trainer.html - страница с работающим тренажером
* trainer.js - файл скриптов для interactive-trainer.html
